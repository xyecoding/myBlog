1. 梯度优化中的吸引子：从数学基础到控制视角

1.1. 1. 引言：为什么需要理解吸引子

在机器学习和优化领域，我们每天都在使用梯度下降及其变体。但你是否思考过：

为什么梯度下降会收敛到某个点？它一定会收敛吗？什么决定了它收敛到哪里？

理解**吸引子（Attractor）**的概念，能帮助我们回答这些问题。吸引子是动力系统中的一种基本现象，描述系统状态随时间演化最终趋向的特定区域或点。

在优化中，吸引子对应着局部最小值（或某些情况下的鞍点）。掌握吸引子的数学条件，不仅能帮助我们设计更好的优化算法，还能理解为何某些训练过程会失败（如陷入局部最优、发散或震荡）。

1.2. 2. 动力系统基础：什么是吸引子

1.2.1. 2.1 动力系统的定义

一个连续时间动力系统由微分方程描述：

x˙=f(x),x∈Rn\dot{x} = f(x), \quad x \in \mathbb{R}^n x˙=f(x),x∈Rn

其中 f:Rn→Rnf: \mathbb{R}^n \to \mathbb{R}^nf:Rn→Rn 是向量场，x˙=dxdt\dot{x} = \frac{dx}{dt}x˙=dtdx​ 表示状态随时间的变化率。

1.2.2. 2.2 平衡点（Equilibrium Point）

定义：点 x∗∈Rnx^* \in \mathbb{R}^nx∗∈Rn 称为平衡点，如果：

f(x∗)=0f(x^*) = 0 f(x∗)=0

直观理解：系统到达 x∗x^*x∗ 后不再演化，"停"在那里。

1.2.3. 2.3 吸引子的严格定义

定义（吸引子）：集合 A⊂Rn\mathcal{A} \subset \mathbb{R}^nA⊂Rn 称为吸引子，如果满足：

  1. 不变性：从 A\mathcal{A}A 内出发的轨迹永远留在 A\mathcal{A}A 内

  2. 吸引性：存在 A\mathcal{A}A 的邻域 U\mathcal{U}U，使得对所有 x0∈Ux_0 \in \mathcal{U}x0​∈U：

    lim⁡t→∞ϕ(t,x0)∈A\lim_{t \to \infty} \phi(t, x_0) \in \mathcal{A} t→∞lim​ϕ(t,x0​)∈A

    其中 ϕ(t,x0)\phi(t, x_0)ϕ(t,x0​) 表示从 x0x_0x0​ 出发、时刻 ttt 的状态

  3. 极小性：A\mathcal{A}A 没有真子集也满足上述条件

吸引域（Basin of Attraction）：所有最终收敛到 A\mathcal{A}A 的初始点集合：

B(A)={x0∈Rn:lim⁡t→∞ϕ(t,x0)∈A}\mathcal{B}(\mathcal{A}) = \{x_0 \in \mathbb{R}^n : \lim_{t \to \infty} \phi(t, x_0) \in \mathcal{A}\} B(A)={x0​∈Rn:t→∞lim​ϕ(t,x0​)∈A}

1.3. 3. 梯度下降的动力学

1.3.1. 3.1 从优化到动力学

给定目标函数 f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R，梯度下降的连续形式为：

x˙=−∇f(x)\dot{x} = -\nabla f(x) x˙=−∇f(x)

这是一个特殊的动力系统，其中向量场 f(x)=−∇f(x)f(x) = -\nabla f(x)f(x)=−∇f(x) 是某个标量函数的负梯度。

关键性质：这种系统称为梯度流（Gradient Flow），具有特殊的动力学行为。

1.3.2. 3.2 能量递减性质

定义"能量" V(x)=f(x)V(x) = f(x)V(x)=f(x)，则：

ddtf(x(t))=∇f⋅x˙=∇f⋅(−∇f)=−∥∇f∥2≤0\frac{d}{dt}f(x(t)) = \nabla f \cdot \dot{x} = \nabla f \cdot (-\nabla f) = -\|\nabla f\|^2 \leq 0 dtd​f(x(t))=∇f⋅x˙=∇f⋅(−∇f)=−∥∇f∥2≤0

结论：f(x(t))f(x(t))f(x(t)) 随时间单调递减（除非到达平衡点）。

这使得梯度下降成为耗散系统，能量不断耗散直至到达局部最小值。

1.4. 4. 吸引子的严格数学条件

现在回答核心问题：什么条件下点 aaa 是梯度下降的吸引子？

1.4.1. 4.1 必要条件：平衡点

定理：aaa 是梯度下降的吸引子 ⇒\Rightarrow⇒ ∇f(a)=0\nabla f(a) = 0∇f(a)=0

证明：若 ∇f(a)≠0\nabla f(a) \neq 0∇f(a)​=0，则 x˙∣x=a=−∇f(a)≠0\dot{x}\big|_{x=a} = -\nabla f(a) \neq 0x˙∣∣​x=a​=−∇f(a)​=0，系统不会在 aaa 处停留，aaa 不是平衡点，更不可能是吸引子。

满足 ∇f(a)=0\nabla f(a) = 0∇f(a)=0 的点称为临界点（Critical Point），包括：

  • 局部最小值
  • 局部最大值
  • 鞍点

1.4.2. 4.2 充分条件：局部严格凸性

定理：若 ∇f(a)=0\nabla f(a) = 0∇f(a)=0 且 Hessian 矩阵 H(a)=∇2f(a)H(a) = \nabla^2 f(a)H(a)=∇2f(a) 正定，则 aaa 是局部渐近稳定的吸引子。

证明（Lyapunov方法）：

取 V(x)=f(x)−f(a)V(x) = f(x) - f(a)V(x)=f(x)−f(a) 作为Lyapunov函数：

  1. 正定性：由于 H(a)≻0H(a) \succ 0H(a)≻0，aaa 是严格局部极小值，存在邻域使 V(x)>0V(x) > 0V(x)>0 for x≠ax \neq ax​=a，且 V(a)=0V(a) = 0V(a)=0

  2. 负定导数：V˙=−∥∇f∥2<0\dot{V} = -\|\nabla f\|^2 < 0V˙=−∥∇f∥2<0 for x≠ax \neq ax​=a

根据Lyapunov稳定性定理，aaa 是渐近稳定的。

1.4.3. 4.3 一维情况的简洁表达

对于 f:R→Rf: \mathbb{R} \to \mathbb{R}f:R→R：

条件 数学表达 结论

退化情况 f′(a)=0f'(a) = 0f′(a)=0 且 f′′(a)=0f''(a) = 0f′′(a)=0 需更高阶分析

示例：

  • f(x)=x2f(x) = x^2f(x)=x2：x=0x=0x=0 处 f′=0,f′′=2>0f'=0, f''=2>0f′=0,f′′=2>0 → 吸引子 ✓

1.5. 5. 高维空间与Hessian几何

1.5.1. 5.1 Hessian矩阵的特征值分解

在临界点 aaa 处，Hessian H(a)H(a)H(a) 是实对称矩阵，可对角化：

H=QΛQT,Λ=diag(λ1,λ2,...,λn)H = Q \Lambda Q^T, \quad \Lambda = \text{diag}(\lambda_1, \lambda_2, ..., \lambda_n) H=QΛQT,Λ=diag(λ1​,λ2​,...,λn​)

其中 λi\lambda_iλi​ 为特征值，QQQ 的列为正交特征向量。

1.5.2. 5.2 临界点的分类

Hessian特征值 几何形状 稳定性
全 λi>0\lambda_i > 0λi​>0 局部凸（碗状） 局部最小值（吸引子） ✓
全 λi<0\lambda_i < 0λi​<0 局部凹（倒碗） 局部最大值（排斥子）✗
有正有负 鞍形（马鞍） 不稳定（鞍点）✗
有零特征值 退化/平坦方向 需进一步分析

1.5.3. 5.3 梯度下降的局部线性化

在 aaa 附近令 x=a+δx = a + \deltax=a+δ，线性化：

δ˙=−∇f(a+δ)≈−H(a)δ\dot{\delta} = -\nabla f(a+\delta) \approx -H(a)\delta δ˙=−∇f(a+δ)≈−H(a)δ

解为 δ(t)=e−Htδ(0)=Qe−ΛtQTδ(0)\delta(t) = e^{-Ht}\delta(0) = Q e^{-\Lambda t} Q^T \delta(0)δ(t)=e−Htδ(0)=Qe−ΛtQTδ(0)

收敛速率：由最小特征值 λmin⁡\lambda_{\min}λmin​ 决定，时间常数 τ∼1/λmin⁡\tau \sim 1/\lambda_{\min}τ∼1/λmin​

条件数问题：若 λmax⁡≫λmin⁡\lambda_{\max} \gg \lambda_{\min}λmax​≫λmin​，收敛慢（病态条件）。

1.6. 6. 吸引域与全局收敛

1.6.1. 6.1 吸引域的复杂性

即使 aaa 是吸引子，也只有从吸引域 B(a)\mathcal{B}(a)B(a) 内出发才能收敛到 aaa。

示例：f(x)=x4−2x2f(x) = x^4 - 2x^2f(x)=x4−2x2

  • 临界点：f′(x)=4x3−4x=0⇒x=−1,0,1f'(x) = 4x^3 - 4x = 0 \Rightarrow x = -1, 0, 1f′(x)=4x3−4x=0⇒x=−1,0,1
  • f′′(−1)=8>0f''(-1) = 8 > 0f′′(−1)=8>0，f′′(1)=8>0f''(1) = 8 > 0f′′(1)=8>0 → 两个局部最小（吸引子）
  • f′′(0)=−4<0f''(0) = -4 < 0f′′(0)=−4<0 → 局部最大（排斥子）

吸引域：

  • B(−1)=(−∞,0)\mathcal{B}(-1) = (-\infty, 0)B(−1)=(−∞,0)
  • B(1)=(0,+∞)\mathcal{B}(1) = (0, +\infty)B(1)=(0,+∞)
  • x=0x=0x=0 是分界线（稳定流形）

1.6.2. 6.2 全局收敛的条件

严格凸函数：若 fff 全局严格凸（H(x)≻0H(x) \succ 0H(x)≻0 for all xxx），则：

  • 存在唯一临界点（全局最小值）
  • 吸引域为整个空间 Rn\mathbb{R}^nRn
  • 梯度下降从任意初值收敛

非凸优化：深度学习中的损失函数通常非凸，存在：

  • 多个局部最小值（多个吸引子）
  • 鞍点（数量通常远多于局部最小值）
  • 平坦区域（近似退化）

1.7.2. 7.2 各分量的作用

分量 优化对应 对吸引子的影响
比例（P） 标准梯度 决定吸引子位置（仍需 ∇f=0\nabla f = 0∇f=0）
积分（I） 动量/自适应 消除稳态误差，对抗噪声，可能改变吸引域
微分（D） 二阶信息/阻尼 抑制振荡，改善收敛速度，影响稳定性边界

1.7.3. 7.3 PID对稳定性的扩展

纯梯度下降的局限：

  • 学习率需满足 η<2/λmax⁡\eta < 2/\lambda_{\max}η<2/λmax​ 以保证稳定
  • 在病态条件下收敛极慢

PID的优势：

  • 积分项可"记忆"历史梯度，帮助穿越平坦区域
  • 微分项提供"预测"，减少在吸引子附近的震荡
  • 适当整定可扩大稳定区域，甚至稳定原本不稳定的点（需谨慎）

注意：PID引入后，平衡点仍需 ∇f=0\nabla f = 0∇f=0，但稳定性条件变为特征值分析闭环系统，参数选择不当可能导致失稳或极限环。

1.8. 8. 实际案例与可视化

1.8.1. 8.1 示例：Rosenbrock函数

f(x,y)=(a−x)2+b(y−x2)2f(x,y) = (a-x)^2 + b(y-x^2)^2 f(x,y)=(a−x)2+b(y−x2)2

  • 全局最小值在 (a,a2)(a, a^2)(a,a2)，位于狭长平坦山谷底部
  • 条件数极差，标准梯度下降收敛极慢
  • 带 momentum（类I项）或自适应学习率的方法显著改善

1.8.2. 8.2 深度学习中的吸引子现象

**损失景观（Loss Landscape）**研究揭示：

  • 局部最小值通常具有相近的损失值（并非坏事）
  • 鞍点更成问题：梯度下降会缓慢穿越
  • 大批量训练倾向于收敛到"尖锐"最小值（吸引域小，泛化差），小批量到"平坦"最小值（吸引域大，泛化好）

1.9. 9. 总结与延伸阅读

1.9.1. 核心要点

  1. 吸引子三要素：平衡点（一阶为零）+ 局部凸性（二阶正定）+ 吸引域内初值

  2. 几何直觉：梯度下降沿最陡下降方向"滑向"能量碗底，碗的曲率决定滑动速度

  3. 高维复杂性：鞍点和退化方向使分析复杂，但Hessian特征值提供局部完整信息

  4. 控制增强：PID类方法通过引入记忆和阻尼，可塑造更有效的吸引动力学

1.9.2. 延伸阅读

  • 经典教材：Khalil《Nonlinear Systems》（Lyapunov稳定性）
  • 优化视角：Nesterov《Introductory Lectures on Convex Optimization》
  • 深度学习：Goodfellow et al. 《Deep Learning》第8章（优化）
  • 前沿研究：Loss Landscape papers（如 Li et al., “Visualizing the Loss Landscape of Neural Nets”, NIPS 2018）

3. KL (Kullback-Leibler) Divergence

3.1. Some Concepts

3.1.1. Self-Information

It can be thought of as an alternative way of expressing probability, much like odds or log-odds, but which has particular mathematical advantages in setting of information theory. It can be interpreted as quantifying the level of ‘surprise’ of a particular outcome. As it is such a basic quantify, it also appears in several other settings, such as the length of a message needed to transmit the event given an optimal source coding of random variable.

3.1.1.1. Two Axiom

  1. A thing which more likely to happen contains less information. The thing definitely happen contains no information.
  2. Two things happen independently contains the information which is the sum of their information.

Thus the Self-Information is defined as

I(x)=−log⁡P(x)=log⁡1P(x)I(x) = - \log P(x) = \log \frac 1{P(x)} I(x)=−logP(x)=logP(x)1​

The Expectation of Self-Information

H(P)=∑xP(x)log⁡(1P(x))H(P) = \sum_x P(x) \log(\frac 1{P(x)}) H(P)=x∑​P(x)log(P(x)1​)

3.3. Cross-Entropy

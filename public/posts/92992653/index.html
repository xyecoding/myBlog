1. 振幅和强度

振幅指复振幅（Complex Amplitude）， 它同时包含幅值和相位信息， 是描述光场完整状态的复数，无法直接测量；

强度（Intensity）是复振幅的模（幅值）的平方， 对应探测器实际响应的光功率密度， 因此可以测量。

2. 时间频率和空间频率

时间频率（光频）： 描述光波随时间振荡的快慢，对于单色光，时间频率是常数。

空间频率：描述光强/振幅在空间上的周期性变化，刻画图像的细节精细度。

光栅：线条间距 0.1 mm → 空间频率 = 10 lp/mm， 包含多种空间频率（边缘=高频，均匀区域=低频）

对不同空间频率的传递能力不同（光学传递函数OTF描述）。

时间频率是光的颜色（振荡快慢）， 空间频率是图像的纹理粗细（空间变化快慢），两者独立。

3. 光的传播方向与相位的关系

3.1. 三维平面波的数学形式

沿任意方向传播的平面波电场：

E(r,t)=E0ei(k⋅r−ωt)\mathbf{E}(\mathbf{r},t) = \mathbf{E}_0 e^{i(\mathbf{k}\cdot\mathbf{r} - \omega t)} E(r,t)=E0​ei(k⋅r−ωt)

其中：

  • r=(x,y,z)\mathbf{r} = (x,y,z)r=(x,y,z) 是位置矢量
  • k=(kx,ky,kz)\mathbf{k} = (k_x, k_y, k_z)k=(kx​,ky​,kz​) 是波矢（常矢量）

3.1.1. 相位函数

空间相位部分为：

ϕ(r)=k⋅r=kxx+kyy+kzz\phi(\mathbf{r}) = \mathbf{k}\cdot\mathbf{r} = k_x x + k_y y + k_z z ϕ(r)=k⋅r=kx​x+ky​y+kz​z

（时间部分 −ωt-\omega t−ωt 对空间梯度无贡献）

3.1.2. 计算相位梯度

对 ϕ(r)\phi(\mathbf{r})ϕ(r) 求梯度：

∇ϕ=(∂∂x,∂∂y,∂∂z)(kxx+kyy+kzz)\nabla\phi = \left(\frac{\partial}{\partial x}, \frac{\partial}{\partial y}, \frac{\partial}{\partial z}\right)(k_x x + k_y y + k_z z) ∇ϕ=(∂x∂​,∂y∂​,∂z∂​)(kx​x+ky​y+kz​z)

逐项求导：

∇ϕ=(kx,ky,kz)=k\nabla\phi = \left(k_x, k_y, k_z\right) = \mathbf{k} ∇ϕ=(kx​,ky​,kz​)=k

结果一目了然：相位梯度 ∇φ 就是波矢 k 本身！

3.1.3. 等相面与传播方向

  • 等相面方程：ϕ=k⋅r=常数\phi = \mathbf{k}\cdot\mathbf{r} = \text{常数}ϕ=k⋅r=常数

    • 这是一个平面，法向量就是 k\mathbf{k}k

  • 梯度方向：∇φ = k 垂直于等相面

  • 传播方向：能量沿 s^=k∣k∣\hat{\mathbf{s}} = \frac{\mathbf{k}}{|\mathbf{k}|}s^=∣k∣k​ 流动

3.1.4. 具体数值例子

设波沿 (1,1,0)(1,1,0)(1,1,0) 方向传播，在真空中波长 λ=500 nm\lambda = 500\,\text{nm}λ=500nm：

  • 波矢大小：k=2π/λ=1.26×107 m−1k = 2\pi/\lambda = 1.26\times10^7\,\text{m}^{-1}k=2π/λ=1.26×107m−1
  • 归一化方向：s^=12(1,1,0)\hat{\mathbf{s}} = \frac{1}{\sqrt{2}}(1,1,0)s^=2​1​(1,1,0)
  • 波矢分量：k=ks^=k2(1,1,0)\mathbf{k} = k\hat{\mathbf{s}} = \frac{k}{\sqrt{2}}(1,1,0)k=ks^=2​k​(1,1,0)
  • 相位：ϕ=k2(x+y)\phi = \frac{k}{\sqrt{2}}(x + y)ϕ=2​k​(x+y)

梯度计算：

∇ϕ=k2(1,1,0)=k\nabla\phi = \frac{k}{\sqrt{2}}(1,1,0) = \mathbf{k} ∇ϕ=2​k​(1,1,0)=k

对于平面波，相位梯度 ∇φ 直接等于波矢 k，其方向完美决定了光的传播方向。

4. 瞳函数、点扩散函数和光学传递函数

4.1. 瞳函数（Pupil Function）（复数域，包含幅值和相位）

P(x)=w(x)exp⁡[2πiϕ(x)]P(x) = w(x) \exp[2\pi i\phi(x)] P(x)=w(x)exp[2πiϕ(x)]

  • xxx：瞳面空间坐标
  • w(x)w(x)w(x)：振幅（孔径内为1，孔径外为0）
  • ϕ(x)\phi(x)ϕ(x)：波前相位误差

哈特曼探测器就是在测量瞳函数的相位梯度

测量对象 哈特曼探测器
瞳函数振幅 w(x)w(x)w(x) 子孔径总光强（积分）
瞳函数相位 ϕ(x)\phi(x)ϕ(x) 局部斜率（梯度）→ 积分重建，间接测量相位

h(ξ)=∣∫P(x)exp⁡(−2πiξx) dx∣2h(\xi) = \left|\int P(x) \exp(-2\pi i \xi x) \, dx\right|^{2} h(ξ)=∣∣∣∣​∫P(x)exp(−2πiξx)dx∣∣∣∣​2

  • ξ\xiξ：像面空间坐标
  • 公式含义：瞳函数的傅里叶变换取模平方

实际上，哈特曼探测器和在焦相机就是在光路里面实现了对瞳函数（变形镜 处） 和点扩散函数的测量， 成像透镜就是实现了瞳函数到点扩散函数之间的傅里叶变换。

4.3. 光学传递函数（Optical Transfer Function, OTF）(复数域)

Ho(f)=No∫P(x)P∗(x−f) dxH_o(f) = N_o \int P(x)P^{*}(x-f) \, dx Ho​(f)=No​∫P(x)P∗(x−f)dx

  • fff：空间频率

  • NoN_oNo​：归一化常数

  • 公式含义：瞳函数的自相关函数（其实就是点扩散函数的傅里叶变换，因为根据维纳-辛钦定理，点扩散函数是瞳函数傅里叶变换的模平方，对其再作傅里叶变换就等价于瞳函数的自相关运算）

  • 模 ∣Ho(f)∣|H_o(f)|∣Ho​(f)∣ → 调制传递函数（MTF）（实数，描述对比度衰减）

  • 相位角 arg⁡(Ho(f))\arg(H_o(f))arg(Ho​(f)) → 相位传递函数（PTF）（描述相位偏移）

4.3.1. 光学传递函数的物理意义

任何线性不变系统（光学系统是其一）满足：

正弦输入 → 同频正弦输出，仅幅度/相位改变

  1. 输入：一个正弦光栅（条纹强度按正弦规律变化）
    数学形式：Iin(x)=I0[1+cos⁡(2πξ0x)]I_{\text{in}}(x) = I_0 [1 + \cos(2\pi\xi_0 x)]Iin​(x)=I0​[1+cos(2πξ0​x)]
    ξ0\xi_0ξ0​ 就是其空间频率

  2. 系统响应：该光栅经过光学系统后

    • 仍是同一频率的正弦条纹（线性系统保频特性）
    • 但对比度下降（幅度衰减）
    • 位置偏移（相位变化）

输出：

Iout(x)=I0[1+∣H(ξ0)∣cos⁡(2πξ0x+arg⁡H(ξ0))]I_{\text{out}}(x) = I_0 \left[1 + |H(\xi_0)| \cos\left(2\pi\xi_0 x + \arg H(\xi_0)\right)\right] Iout​(x)=I0​[1+∣H(ξ0​)∣cos(2πξ0​x+argH(ξ0​))]

  • ∣H(ξ0)∣|H(\xi_0)|∣H(ξ0​)∣ ：该频率下的对比度衰减系数 → MTF
  • arg⁡H(ξ0)\arg H(\xi_0)argH(ξ0​)：该频率下的相位偏移 → PTF

4.3.1.1. 点扩散函数的作用

  1. PSF 是系统的脉冲响应：点光源 δ(x)\delta(x)δ(x) 的成像结果

    h(x)=系统{δ(x)}h(x) = \text{系统}\{\delta(x)\} h(x)=系统{δ(x)}

  2. 卷积定理：任意输入 Iin(x)I_{\text{in}}(x)Iin​(x) 的输出是

    Iout(x)=Iin(x)∗h(x)I_{\text{out}}(x) = I_{\text{in}}(x) * h(x) Iout​(x)=Iin​(x)∗h(x)

    （∗*∗ 表示卷积）

  3. 频域关系：对两边做傅里叶变换

    F{Iout}=F{Iin}⋅F{h}\mathcal{F}\{I_{\text{out}}\} = \mathcal{F}\{I_{\text{in}}\} \cdot \mathcal{F}\{h\} F{Iout​}=F{Iin​}⋅F{h}

  4. 定义OTF：

    H(ξ)=F{h(x)}H(\xi) = \mathcal{F}\{h(x)\} H(ξ)=F{h(x)}

点扩散函数是系统在空域的脉冲响应，其傅里叶变换OTF就是系统在频域的频率响应函数，直接给出每个空间频率正弦条纹的对比度衰减和相位偏移量，因此完整表征了系统的传递能力。

5. 透镜与傅里叶变换

波前的空间频率分量对应着不同传播角度的平面波， 透镜将这些角度信息变换为焦平面的位置信息，因此透镜相当于把空间频率直接映射为位置信息，实现了傅里叶变换。

相位梯度正比于局部空间频率（或波前倾斜角）， 因此哈特曼探测器通过测量子孔径的角度信息直接获得相位梯度，进而积分重建瞳函数的相位分布。
